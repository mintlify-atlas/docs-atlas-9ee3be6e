title: "Individual Global Leaderboard"
description: "Get individual leaderboards across all raids for various categories"
---

## Endpoint

<CodeGroup>
```bash GET
GET /leaderboard/individual/global/{category}
```
</CodeGroup>

Individual leaderboards across all raids. This endpoint returns rankings for players based on their cumulative performance across all raid activities.

## Path Parameters

<ParamField path="category" type="string" required>
  The leaderboard category to retrieve. Must be one of:
  - `clears` - Total raid completions
  - `full-clears` - Total full (fresh) raid completions
  - `sherpas` - Total sherpa runs (helping first-time raiders)
  - `speedrun` - Fastest completion times (duration format)
  - `world-first-rankings` - World first race performance scores
  - `in-raid-time` - Total time spent in raids
</ParamField>

## Query Parameters

<ParamField query="count" type="integer" default="50">
  Number of entries to return per page. Must be between 10 and 100.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number of leaderboard data. Ignored if `search` is provided. Must be greater than 0.
</ParamField>

<ParamField query="search" type="string">
  Search for a specific player by membershipId. Returns results centered around that player's position. Pattern: `^\d+n?$`
</ParamField>

## Response

The response varies based on the category. Some categories return individual entries, while others return team entries.

<ResponseField name="type" type="string">
  Either `"individual"` or `"team"` depending on the category
</ResponseField>

<ResponseField name="format" type="string">
  Either `"duration"` or `"numerical"` indicating the value format
</ResponseField>

<ResponseField name="page" type="integer">
  Current page number
</ResponseField>

<ResponseField name="count" type="integer">
  Number of entries returned
</ResponseField>

<ResponseField name="entries" type="array">
  Array of leaderboard entries
  
  <Expandable title="Individual Entry Properties">
    <ResponseField name="position" type="integer">
      Leaderboard position (1-indexed)
    </ResponseField>
    
    <ResponseField name="rank" type="integer">
      Rank accounting for ties
    </ResponseField>
    
    <ResponseField name="value" type="number">
      The metric value (duration in seconds for speedrun, count for others)
    </ResponseField>
    
    <ResponseField name="playerInfo" type="object">
      Player information object
      
      <Expandable title="PlayerInfo Properties">
        <ResponseField name="membershipId" type="string">
          Destiny 2 membership ID
        </ResponseField>
        
        <ResponseField name="membershipType" type="integer">
          Platform type (1=Xbox, 2=PlayStation, 3=Steam, 6=Epic)
        </ResponseField>
        
        <ResponseField name="bungieGlobalDisplayName" type="string">
          Bungie display name
        </ResponseField>
        
        <ResponseField name="bungieGlobalDisplayNameCode" type="string">
          Bungie name code (4 digits)
        </ResponseField>
        
        <ResponseField name="iconPath" type="string">
          Path to player's icon on Bungie CDN
        </ResponseField>
        
        <ResponseField name="lastSeen" type="string">
          ISO 8601 timestamp of last activity
        </ResponseField>
        
        <ResponseField name="isPrivate" type="boolean">
          Whether the player's profile is private
        </ResponseField>
        
        <ResponseField name="cheatLevel" type="integer">
          Cheat detection level (0=None, 1=Suspicious, 2=Moderate, 3=Extreme, 4=Blacklisted)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl --request GET \
  --url 'https://api.raidhub.io/leaderboard/individual/global/clears?count=10&page=1' \
  --header 'X-API-KEY: your_api_key'
```

## Error Responses

<ResponseField name="PlayerNotOnLeaderboardError" type="404">
  Returned when searching for a player who is not on the leaderboard
  
  ```json
  {
    "minted": "2024-03-15T12:00:00.000Z",
    "success": false,
    "code": "PlayerNotOnLeaderboardError",
    "error": {
      "membershipId": "4611686018488107374"
    }
  }
  ```
</ResponseField>

<ResponseField name="PathValidationError" type="404">
  Invalid category parameter
</ResponseField>

<ResponseField name="QueryValidationError" type="400">
  Invalid query parameters (count, page, or search)
</ResponseField>
