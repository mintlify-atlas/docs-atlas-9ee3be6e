## What is RaidHub API?

RaidHub API is a powerful RESTful API that provides access to comprehensive Destiny 2 raid statistics, leaderboards, and player performance data. Built for developers who want to integrate raid analytics into their applications, websites, or tools.

Our API aggregates and processes data from millions of raid completions, providing:

- Detailed player raid statistics and history
- Global and raid-specific leaderboards
- Real-time instance and PGCR (Post-Game Carnage Report) data
- Clan raid statistics
- Weapon usage metrics and population analytics

## API Endpoint

All API requests should be made to:

```
https://api.raidhub.io
```

<Note>
  The API requires an API key for production use. Development requests (from localhost) do not require authentication.
</Note>
